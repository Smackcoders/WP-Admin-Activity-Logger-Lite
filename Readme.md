# Scope: Admin Activity Logger Lite

## Plugin Name

Admin Activity Logger Lite

## Plugin Purpose

Admin Activity Logger Lite tracks important admin-side changes in WordPress for accountability, troubleshooting, and client-site maintenance.

The plugin should provide a lightweight audit trail showing who changed what, when it happened, and what action was performed.

## Target Users

- Multi-admin websites
- Agencies
- Client websites
- Membership platforms
- WordPress maintenance teams
- Website owners
- Support teams
- Admin teams
- Security-conscious site managers

## MVP Scope

The free community version should include:

- Log post create/update/delete
- Log page create/update/delete
- Log plugin activation/deactivation
- Log user creation/deletion
- Log settings changes
- Log basic media upload/delete
- Admin activity log page
- Filter logs by user/action/date
- Search logs
- Export logs as CSV
- Clear old logs
- Basic settings page
- Admin dashboard summary
- Security with nonce, capability checks, sanitization, escaping
- Translation-ready strings
- Basic PHPUnit/WP test coverage

## Future Pro Scope

- Real-time alerts
- WooCommerce order activity logs
- Role-based log visibility
- External log storage
- Slack/Telegram/Google Chat notifications
- GitLab issue creation for critical admin events
- Multisite network-wide activity logs
- Advanced diff view for content/settings changes
- AI-based suspicious activity summary
